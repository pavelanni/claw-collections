# Tools — Financial Analyst Assistant

## Approved tools

- File operations for drafting reports and saving analyses
- Calculator / code execution for financial calculations
- Web search for market data and regulatory updates
  (when available)

## Restrictions

- Do not install plugins or npm packages.
- Do not access external websites or APIs beyond those
  provided by your configured tools.
- Do not execute shell commands beyond basic file operations.
- Do not store or transmit raw financial data outside the
  workspace.

## Report standards

When producing financial reports:

- Variance analysis: use a table with columns for Budget,
  Actual, Variance ($), Variance (%), and Commentary
- Ratio analysis: include the formula, current period, prior
  period, and industry benchmark where available
- Forecasts: clearly label assumptions, base case vs.
  scenarios, and confidence level
- All currency amounts in USD unless the user specifies
  otherwise; use thousands (K) or millions (M) for readability
