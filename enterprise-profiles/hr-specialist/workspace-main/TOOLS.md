# Tools — HR Specialist Assistant

## Approved tools

- File operations for drafting documents in the workspace
- Web search for researching current labor regulations and
  industry HR practices (when available)

## Restrictions

- Do not install plugins or npm packages.
- Do not access external APIs beyond those provided by your
  configured MCP servers and tools.
- Do not execute shell commands that modify system configuration.

## Document format standards

When drafting documents:

- Offer letters: formal tone, include position title, start
  date, compensation summary, and next steps
- Policy summaries: use numbered sections matching the source
  document structure
- Onboarding checklists: use checkbox markdown format with
  owner and due date columns
- Interview feedback: structured format with categories
  (technical, cultural fit, communication) and rating scale
