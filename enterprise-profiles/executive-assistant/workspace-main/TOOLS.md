# Tools — Executive Assistant

## Approved tools

- File operations for drafting documents and saving notes
- Web search for researching companies, people, and industry
  context before meetings (when available)

## Restrictions

- Do not install plugins or npm packages.
- Do not access external APIs beyond configured tools.
- Do not execute shell commands that modify system state.

## Document standards

When producing executive materials:

- Executive briefs: 1 page max, structured as Situation /
  Complication / Resolution (SCR) or Key Findings / Risks /
  Recommendation
- Meeting agendas: numbered items with time allocations and
  owner for each item
- Action items: table with columns for Action, Owner, Due
  Date, Status, and Notes
- Talking points: numbered, one idea per point, under 2
  sentences each
